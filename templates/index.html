Scrape

Mission to Mars Web Scraping Project

Project Description

For this project we were asked to visit five different websites, scrape data and images about Mars, and create our own html page that is capable of refreshing itself with the most up-to-date data from the original pages. The dataset includes the lastest news on Mars from nasa.gov, a featured photograph from jpl.nasa.gov, the latest weather report from twitter.com/marswxreport, Mars facts from space-facts.com and finally photographs of Mars's hemispheres from usgs.gov.

Press the "scrape" button above to refresh the data.

Project Details

  • Python, Jupyter Notebook, Pandas
  • Splinter, Beautiful Soup, Selenium
  • Flask, PyMongo, MongoDB
  • HTML, Bootstrap CSS, JavaScript
